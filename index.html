Acerca de

Progressive web apps (PWA) o aplicación web progresiva es una solución basada en la web tradicional que todos conocemos, aunque incorpora algunas particularidades que la hacen parecerse a una app nativa para teléfonos móviles y tabletas.

App web =/ PWA

No todas las aplicaciones web pueden llamarse Progressive web apps. Las PWA reales consisten en un protocolo HTTPS encriptado, uno o varios service workers, un archivo de manifiesto y un tiempo de carga rápido debido a su arquitectura central.

Protocolo HTTPS

Cualquier aplicación web debe ser un sitio seguro y navegar a través de una red segura.

Service Workers

Los trabajadores del servicio son scripts que controlan la forma en que un navegador web maneja las solicitudes de red y el almacenamiento en caché de activos

Manifest

El archivo de manifiesto es un archivo JSON. Su función principal es controlar la forma en que una aplicación aparece para los usuarios finales.
